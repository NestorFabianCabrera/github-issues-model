Dos tipos de entrada

El modelo combina señales de texto y metadatos estructurados del issue.

  • TTexto: título (×2) + descripción → TF-IDF 8,000 términos, bigramas
  • NNuméricas: has_assignee, has_milestone, comments, label_count
  • NTemporales: day_of_week, hour_of_day
  • NTexto derivado: has_code_block, has_url, has_question, title_len

Código

El pipeline en Python

  • 1TF-IDF con sublinear_tf y stop words
  • 2StandardScaler para normalizar features numéricas
  • 3hstack() combina matrices sparse eficientemente
  • 4Random Forest con class_weight='balanced'
train_model.py
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.ensemble import RandomForestClassifier
from scipy.sparse import hstack, csr_matrix

tfidf = TfidfVectorizer(
    max_features=8000,
    ngram_range=(1, 2),
    sublinear_tf=True,
    stop_words="english",
)
X_tfidf = tfidf.fit_transform(X_train["text"])
X_num   = scaler.fit_transform(X_train[NUM_FEATURES])
X_final = hstack([X_tfidf, csr_matrix(X_num)])

rf = RandomForestClassifier(
    n_estimators=300,
    class_weight="balanced",
    n_jobs=-1,
)
rf.fit(X_final, y_train)
